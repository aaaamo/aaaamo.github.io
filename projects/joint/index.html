Joint

Date October 2024 - Present
Type Personal Work
Collaborators Changyong Kim
Tech Stack Node.js, React.js, Three.js, MongoDB, git

This experimental web-based service aims to streamline architectural detailing by automating and customizing section details through a simple skeleton-line approach. With each new line drawn, joint details are created instantly, and a suite of intuitive editing tools allows users to set priorities, reorder materials, and fine-tune baselines. Although still in progress, the project's future milestones include building a comprehensive database of section details for users to select and customize, as well as integrating an AI model to suggest details based on verbal prompts.
